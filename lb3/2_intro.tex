\chapter*{Цели и задачи}
\addcontentsline{toc}{chapter}{Введение}
\label{ch:intro_sync}

\textbf{Цель лабораторной работы:} изучение принципов построения информационных систем с использованием метрических методов классификации.


\textbf{Основные задачи:}
\begin{itemize}
    \item изучение инструментария Python для реализации алгоритмов метрической классификации;
    \item изучение методов оптимизации параметров метрической классификации;
    \item освоение модификаций kNN-метода.
\end{itemize}
