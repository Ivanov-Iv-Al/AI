\chapter*{Цели и задачи}
\addcontentsline{toc}{chapter}{Введение}
\label{ch:intro_lab6}

\textbf{Цель лабораторной работы:} разработка единого пайплайна для решения задачи одномерной регрессии.

\textbf{Основные задачи:}
\begin{itemize}
    \item реализовать конвейер для выполнения всех стадий обработки данных при решении задачи одномерной регрессии;
    \item получение теоретических представлений о задаче регрессии;
    \item получение навыков использования пайплайна при решении задачи машинного обучения;
    \item получение навыков рефакторинга кода в задачах машинного обучения.
\end{itemize}

В качестве набора данных для выполнения индивидуального задания выбран набор данных \textbf{Abalone} (морское ушко), который содержит информацию о возрасте моллюсков, определяемом по количеству колец на раковине.
